Ferramentas Avançadas do Banco

Este script varre o banco de dados fundindo números fantasma ou mal formatados ao contato correto.

Preenche o campo "filial_name" em todos os setores existentes para o bot identificar corretamente.

Varre o banco de dados e converte todos os contatos com 13 dígitos para 12 (removendo o 9º dígito), compatibilizando com o WAHA.

Converte todos os chats e mensagens de qualquer outra sessão (ex: default, yamar) para a sessão "corpal".

Dispara um alerta push nativo em todos os celulares dos entregadores que permitiram notificações (Mesmo com o app fechado).
